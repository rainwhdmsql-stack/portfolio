# CURIO Portfolio Production Build v1.0

CURIO는 포트폴리오에 내장된 AI 큐레이터가 조현정 디자이너의 실제 포트폴리오 자료를 읽고, 해석하고, 정리한 디지털 기록형 포트폴리오입니다.

## 포함된 주요 개선 사항
- 진입 화면 오류 수정
- Hero 첫 섹션 한영 혼합 타이틀 보정
- Dashboard 카드 비율 및 MAIN ROLE 텍스트 넘침 보정
- CURIO 기록 상태 UI
- 섹션 진입 모션 및 읽기 전 메모 방식
- Project Window 버튼 동작 보정
- End Of Report 카드 및 링크 버튼 구조
- 전체 CSS/JS 구조 유지

## 파일 구조
- index.html
- css/*.css
- js/main.js

## CURIO Language Guide

- System language: English  
  Used for dashboard labels, navigation metadata, report status, card headers, and short interface labels.  
  Example: CURIO LOG, MAIN ROLE, FOCUS, EXPERIENCE, PROJECT, PROJECT TYPE, GENERATED BRIEFING.

- Story language: Korean  
  Used for briefing copy, descriptions, curator notes, section explanations, and portfolio narrative.  
  Example: 사용자가 선택한 주제에 맞춰 관련 프로젝트를 선별하여 안내합니다.

- Principle  
  CURIO should not overuse English. English is reserved for system-like labels, while Korean carries the actual story and explanation.
